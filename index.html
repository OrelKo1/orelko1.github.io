C-Based Assembler for Custom CPU - Systems Programming Lab

This project is a C-based assembler developed for a custom CPU as part of a systems programming lab course. It translates assembly code into machine code through macro expansion, instruction encoding, error validation, and binary file generation. The assembler supports a two-pass compilation process and produces machine code in base64 format.
